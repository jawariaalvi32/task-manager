# TASK MANAGER 

A simple todo list to manage tasks build with redux toolkit

### [Live Demo](https://coronacovid19-tracker.netlify.app/)

## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/jawariaalvi32/corona-tracker.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
### Run
 ```sh
   npm start
   ```
