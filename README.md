# TASK MANAGER 

A simple todo list to manage tasks built using react and redux toolkit

### [Live Demo](https://managetasks.netlify.app/)

## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/jawariaalvi32/task-manager.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
### Run
 ```sh
   npm start
   ```
